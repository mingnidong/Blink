# Project Proposal: Team 10 - Amber, Emily, Mingni, Anika

## Project title: EduPulse

## Summary
- Students press button during class whenever they are confused
- Professor is able to see when people are confused by timestamp/heatmap
- The website also stores recordings until transcription is made.
- Professor can download audio files before deletion

## Scope
### Requirements
- Account creation for teachers
- Visualization of confusion “hotspots” directly on the heatmap
- Databases storing confusion counts, timestamps, and recordings that are deleted after transcription and optional download
### Reach goals
- Transcribe recordings, and give like 2 word summaries for each point that had lots of confusion counts, make it visually show up in the heatmap
- Optional polls professors can create
### Tech stack
- Vercel deploy + supabase database. Free, lightweight
### Testing logistics
- Ask TA friends to use this in their recitations
- Ask professor to use it in class
